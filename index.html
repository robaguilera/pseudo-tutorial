Psuedo and Attribute Selectors

Purpose

The purpose of this class is to teach you to use *CSS to target *specific elements on any given page. See, what we want to do is leverage the power of CSS to target exactly what we want without having to mess with our markup. This allows us to create clean markup which makes it easily readable to programmers as well as make it accessible to screen readers, without tons of crap.
